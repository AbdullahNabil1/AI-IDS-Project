# AI Intrusion Detection System (AI-IDS)

## 📌 Description
This project is an AI-based Intrusion Detection System that detects network attacks using Machine Learning techniques.

## ⚙️ Features
- Data preprocessing
- Training ML models
- Detecting network intrusions
- Evaluating model performance

## 🧠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## ▶️ How to Run
```bash
pip install -r requirements.txt
python main.py
